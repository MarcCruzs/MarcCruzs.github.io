## Homework 1: 
**Inheritance, Polymorphism, Aggregation, Composition (OOP Principles)**
1. **Inheritance:** Create an employee hierarchy with multiple types of employees (Salaried, Hourly, Commission, etc.). Build classes representing these types and instantiate them with example data.
2. **Polymorphism:** Design a `Ship` class with two derived classes: `CruiseShip` and `CargoShip`. Use a polymorphic print function to display specific details for each type.
3. **Aggregation:** Create a `Course` class that "has-a" `Instructor` and `Textbook`. Modify the program to accommodate multiple instructors and textbooks.
4. **Composition:** Design a file system with `Folder` and `File` classes. Implement functions to delete folders and print the entire structure recursively.

## Homework 2:
**Object-Oriented Analysis (OOA) and Design (OOD)**
1. **Part 1:** Use cases to design a navigation application (CPP Maps) covering Navigation, Search, Explore, and location sharing.
2. **Part 2:** Highlight potential classes (nouns) and actions (verbs) from the use cases.
3. **Part 3:** Design CRC cards for identified classes, and create a final graph representing relationships among them.

## Homework 3:
**UML Design**
1. Create UML Class Diagrams for Homework 1 (Parts 1–4).
2. Develop UML Object Diagrams at specific points in Homework 1's lifecycle, including object creation and deletion.

## Homework 4: 
**Creational Design Patterns (Builder, Abstract Factory, Singleton)**
1. **Pizza Chain:** Use the builder pattern to create pizzas with various toppings for Pizza Hut, Little Caesars, and Dominos.
2. **Macronutrient Meals:** Generate meals based on diet plans using Abstract Factory, Factory, and Singleton patterns. Six customers demonstrate different diet restrictions and randomly generated meals.

## Homework 5: 
**Structural Design Patterns (Bridge, Decorator)**
1. **Bridge:** Create a messaging system that uses the Bridge pattern to support multiple messaging apps and message types.
2. **Decorator:** Design a restaurant ordering system where customers can order food items with additional toppings, applying the decorator pattern to calculate costs with and without discounts.

## Homework 6: 
**Behavioral Design Patterns (Memento, Mediator, Iterator)**
1. **Chat Application:** Implement a chat application using Memento to undo messages and Mediator to manage communication between users. Extend functionality with an iterator to browse message history.

## Homework 7:
**Structural Design Patterns (Flyweight, Proxy)**
1. **Text Editor:** Use Flyweight to manage character properties (font, size, color) efficiently in a text editor.
2. **Music Streaming App:** Implement a Proxy pattern to cache song metadata, optimizing access without replaying music.

## Homework 8: 
**Behavioral Design Patterns (State, Chain of Responsibility)**
1. **Vending Machine:** Design a vending machine with states (Idle, Waiting for Money, Dispensing) and a Chain of Responsibility to handle snack dispensing. Use UML diagrams to depict the states and transitions.
